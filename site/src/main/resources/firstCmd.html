Create your first Crash command


It's really simple :

  • Download Crash and unzip it.
  • cd $CRASH_HOME/cmd (You will see other Groovy file)
  • Create a new file with your text editor : test.groovy
  • Write return "It is my first command !!!" in this file

  • cd $CRASH_HOME/bin
  • ./crash.sh
  • Type help at prompt and you will see test command.
  • Type test and you will see : It is my first command !!!

Ok it's a simple command but you can do more complex command. See documentation for more details.You could also go in source directory to see existing command ...
